## Installation

```
$ npm install sonea-utils
```

## Usage

```JavaScript
var soneaUtils = require('sonea-utils');

if(soneaUtils.file.isDirectory(__dirname)) {
  console.log("It is a directory (>^.^)>");
}
```

## utils currently for:

* [debug](https://www.npmjs.com/package/debug)
* file
* global (exposing)
